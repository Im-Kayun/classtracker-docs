## /register

Creates a new user, but without any specific type. Used only for admin creation.

- HTTP Method: `POST`
- Parameters:
    - name: `string`
    - surname: `string`
    - email: `string`
    - password: `string`
- Returns: `JSON`
    - id: `string`
    - email: `string`

## /login

Login in the platform.

- HTTP Method: `POST`
- Parameters:
    - email: `string`
    - password: `string`
- Returns: `JSON`
    - id: `string`
    - email: `string`

## /logout

Logout the current user.

- HTTP Method: `POST`
- Parameters: `None`
- Returns: `Code 200`
